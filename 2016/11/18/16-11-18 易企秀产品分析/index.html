易企秀产品分析

2016-11-18

产品分析 商业模型 产品商业化

易企秀 - 产品概述

场景制作营销工具：是一款针对移动互联网营销的手机幻灯片、H5场景应用制作工具，将原来只能在PC端制作和展示的各类复杂营销方案转移到更为便携和展示的手机上，用户随时随地根据自己的需要在PC端、手机端进行制作和展示，随时随地营销。

分析框架

  1. 商业模型
  2. 产品模型
  3. 运营模型

商业模型（商业思路）

用户群与需求

  1. 普通用户诉求：傻瓜式的使用，满足0成本微营销与亲友互动的需求
  2. 中小企业用户诉求：节省设计、开发等人力成本。将各个部门相关营销需求进行快速的落地和传播。
  3. 企业宣传、产品介绍、会议邀请、活动报名、网站导流、节日贺卡、婚礼邀请

    商业目标实现路径

    一、To普通用户

  4. 简单制作页面的产品，在满足普通用户的诉求下，达成高用户覆盖率。打造理想广告营销渠道（详细见产品分析）进行变现；成为营销渠道的输出口
  5. 高基础用户人群使用，达成品牌效应，方便销售团队地推，及打造广告产品线的根基进行变现；

    二、To企业用户

  6. 帮助企业用户节省人力和时间成本，促成企业用户进行付费使用
  7. 为企业用户，筑建场景-做推广-管数据-再营销的移动营销闭环。成为营销渠道的输入口

运营体系

市场情况（标志性事件）：

  • 自2014年，HTML标准协议确定，在国际上H5被极度推崇
  • 国内互联网首推移动手机市场，团队开始抢第一波移动用户红利，移动应用开始爆发，H5的移动端特性凸现
  • 微信在2014-2015的大规模普吉，从一线城市，普遍渗透到二三线城市，移动端社交红利凸现，移动营销需求旺盛

产品运营路径：

  1. 首推覆盖普通用户和付费用户的重叠功能：H5模板制作。对普通用户提供基础编辑功能，对企业付费用户地推定制化模板（达成用户体量目标）
  2. 推进企业用户付费产品的完善
  3. Push推送营销渠道的搭建
  4. 易企传的接单任务方式来增加自身传播的能力

推广方式：

初期

  • 通过社交口碑推广
  • 搜狐焦点家居场景类产品合作伙伴推广；
  • 光明日报采用易企秀，覆盖大量的用户

    中期

  • 各媒体曝光：环球采访
  • 举办各种合作活动：教师节活动（中国教育新闻网合作）

    后期（当前）

  • 成都环绕城市群建立销售地推
  • 电话销售
  • 广告合作
  • 商务拓展

用户数据：

APP终端安装：500万+
PC端1000多万

产品模型

根据商业目标，建立的几个产品线；

一、基础产品：快速制作移动营销的页面

  1. 对场景进行模板式设计（UI设计）
  2. 轻松制作H5页面（前端开发工程师）
  3. 快速的配置与发布服务（云服务功能）

二、产品积分体系

名称：秀点

  1. 获取方式：购买1RMB = 1秀点；
  2. 使用：购买推广服务
  3. 作用：弥补会员体系大额充值的弊端；能够满足小额使用场景需要；同时可以对广告营销费用进行充值支付

    BBS论坛金币

  4. 获取方式：社区基本操作；论坛任务

  5. 使用：兑换秀点（兑换条件：100金币以上开始兑换兑换比例：10金币=1秀点）

三、付费产品体系

会员套餐：详见易企秀介绍页面内容

1.基础服务

  1. 企业专属：子帐号使用；企业样例
  2. 子有品牌展示：去掉易企秀等品牌资阳；去LOGO、自定义域名、联合品牌等
  3. 编辑功能：个人帐号无【填空模式创建、编辑区缩放、添加长页面、禁止滑动翻页、一键导入PSD】；企业免费账户【比个人账户多有一个填充模式创建】；企业三个付费版，具备全部编辑功能
  4. 保障服务：云服务【误删恢复；CDN加速；独立场景服务器】

2.数据统计分析服务

  1. 分享统计
  2. 访客分析
  3. 页面统计
  4. 数据概况
  5. 访问统计
  6. 互动统计

3.场景推广

  1. 扩展网址推广：数据跟踪各个渠道的数据
  2. 短信推广：利用短信推广，让用户付费，来cover掉成本，并同时能产生利润
  3. 社交分享：全部免费
  4. 广告推广：DSP广告（预算5000+的用户适合）CPC、CPM；红人直推

4.人工服务

  1. 专属客服：VIP专属客服电话；VIP转述客服顾问
  2. 培训：基础视频、标准（视频、社群直播、线下）、定制（视频、社群直播、线下、定制）
  3. 场景保障：场景前置审核（优先审核权）；驳回加急审核；审核、关闭的短息提醒

5、销售套餐

  1. 会员套餐
  2. 局部特权套餐
  3. 周边产品
  4. 定制化开发
  5. 秀点套餐购买
  6. APP推送广告服务套餐

四、核心收入产品——易企传

  1. 免费的秀场展示：吸引优秀的案例进行展示：提供其他用户参考的内容源
  2. 举办活动：吸引创作，展示品牌和介绍功能
  3. 易企传任务：订单任务式传播，对内部流量的复用
  4. 资源对接：场景定制

易企秀未来规划

  1. 拓展重庆、长沙、武汉、成都城市群的中小企业的付费用户群（成都的销售团队正在搭建），挖掘二三线城市的可能性
  2. 持续覆盖更多普通免费用户群，为营销渠道扩展
  3. 完善自传播的广告商业化的产品线，挖掘更多广告变现的可能性
