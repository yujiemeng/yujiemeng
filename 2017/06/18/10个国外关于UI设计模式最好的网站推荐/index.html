10个国外关于UI设计模式最好的网站推荐

2017-06-18

User Interface (UI) Design Graphic Design User Interface (UI) Design Patterns



Interaction Design Foundation

Interaction Design Foundation是一个国际大型用户体验设计社区，旨在为设计从业者提供丰富的学习内容和设计知识。


这篇文章是转载Interaction Design Foundation社区：文章已在原社区获得作者的授权。


10 Great Sites for UI Design Patterns



你不想花你的一生重新设计你的设计稿和创意吧？ 不，我们也不想这样。如果您正在寻找设计方案之前，已经在不同应用中使用过此设计方案;那么你应该已经有了设计模板在脑海中了。这是设计模式。

我们已经列出了一些在网络上查找设计模式的最佳场所。现在，虽然所有这些资源都是免费的，但有些可能会要求您为一些额外资源支付;我们只是希望您知道我们不是任何其他网站的附属公司，也没有任何经济利益，您从任何其他网站购买任何东西都可以根据您自身的需要。

UI – Patterns

UI Patterns之所以受到推荐，是因为在这个网站上，您能够找到自己喜欢的设计模型，如果对这些创意方案感觉到有用的话，可以对它们进行分组收藏。他们还有一些关于UI设计模式有趣的文章，可以帮助您扩展您在设计中使用模式的方式。


版权所有: UI Patterns. 版权条款使用: 正常

WELIE Interaction Design Patterns

WELIE不是我们见过的最漂亮的网站，但是有一个非常有用的交互设计模式库，它是逻辑分组的，对很多用户来说，是很有帮助的。

Pattern Tap

Pattern Tap正在关闭10,000个用户生成的模式集，这使其成为UI设计师的令人鼓舞的资源。 下拉菜单系统使您很容易找到您需要的内容。

UI Scraps

在这里，您将看到什么是好的UI设计，什么是糟糕的UI设计和什么是丑陋的UI设计。UI Scraps是一个不错的博客，它将整个界面设计视为一个整体，然后修改所使用的模式。 你可以在这里找到一些很棒的作品，当然也会遇到很糟糕的作品; 所有的内容都需要批判性的看待，所以你能够了解什么样的设计是不起作用的。

Pattern Browser

Pattern Browser确实像它的名字字面含义那样，它可以让您在浏览器窗口中快速切换UI模式。网站里并没有像一些较大网站的资源，尽管如此，它依然能够提供不错的其他资源。

版权所有: Pattern Browser. 版权条款使用: 正常

User Interface Engineering

User Interface Engineering提供的模式多，这是一个很重的资源，可以满足您的很多UI需求。 用户和网站生成的评论是非常宝贵的。

Design Snips

Design Snips 提供给用户一个非常好的图形库，可以将一些图形库放在自己的UI作品中。

CSS Bake

CSS Bake中有你在“烘焙”网站时所需的很好设计元素库。它也是一个很棒的灵感来源地，在网站中有很多确实不错的设计样式可供参看。

Elements of Design

在Elements of Design中有一些设计元素可供使用，通过分组的方式，使它在使用的过程中变得很方便。这有点超出了UI的范畴，但是这里还有很多UI用户也可以使用它。

Web Design Practices

Web Design Practices已经运营有一段时间了，但似乎没有太多更新; 这里有一些非常有用的东西。
